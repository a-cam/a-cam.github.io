PCB - Pressure Sensor Board

For the underwater vehicle that I am building with a student society, there will be a pressure sensor for depth localisation on the vehicle. As such there is a need for a board to provide an interface to the pressure sensor chip which is what I will be developing.

I used the pcb software Kicad to create the following schematics and pcb layouts for the board.

The next steps in this project include getting the pcb manufactured and then the soldering of the surface mount components and chip debugging can start.
